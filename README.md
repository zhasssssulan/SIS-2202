# SIS-2202
# Automated Essay Scoring Using Machine Learning

## Project Overview
This project implements an Automated Essay Scoring (AES) system using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to predict student essay scores automatically based on textual content.

## Course Information
- Course: Machine Learning
- Instructor: Professor Dr. Abdul Razaque, PhD & Postdoc

## Dataset
- Name: ASAP Automated Essay Scoring Dataset
- Source: Kaggle
- Size: ~13,000 essays
- Target variable: domain1_score
